# Stock_marketing_dashboard

A comprehensive stock market analysis dashboard built with Streamlit, featuring interactive visualizations and multi-stock comparisons.

## Features

- **Multi-Stock Analysis**: Compare multiple stocks simultaneously
- **Interactive Charts**: Candlestick, line, bar, and correlation visualizations
- **Time Series Analysis**: Custom date range selection
- **Volume Analysis**: Trading volume pattern analysis
- **Returns Calculation**: Daily and cumulative returns
- **Statistical Summary**: Key performance metrics

## Data Sources

This project uses stock market datasets from Kaggle:
- Apple (AAPL) stock data
- Google (GOOGL) stock data
- Microsoft (MSFT) stock data
